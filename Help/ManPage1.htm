Загрузка информации о дороге из базы данных

Для загрузки информации из базы данных необходимо выбрать пункт меню (Файл)+(Открыть). В окне справочника дорог выбрать дорогу и нажать на кнопку [Ok] Одна дорога может быть открыта в нескольких окнах. В этом случае информация в окнах дублируется и синхронизируется, то есть при изменении информации в одном окне, в другом окне информация также изменится.

Сохранение информации в базе данных

Для сохранения сделанных изменений необходимо выбрать пункт меню (Файл)+(Сохранить). В этом случае будет сохранена информация из активного окна.
