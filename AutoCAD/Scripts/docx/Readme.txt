Допустимые поля для автозамены:

    {RoadName} -- название дороги
    {DistrictName} -- название района
    {NumDataSource} -- номер источника дороги
    {NumRoad} -- номер дороги
    {RoadBegin} -- начало дороги
    {RoadEnd} -- конец дороги
    {RoadBegin_km} -- начало дороги в формате 0+000
    {RoadEnd_km} -- конец дороги в формате 0+000
    
Внутри таблицы можно вставлять SQL скрипты. В таблицу будут добавлены строки содержащие результаты этих скриптов. Для этого в последней строке в первой  ячейки надо вставить следующее содержимое:

//===== НАЧАЛО СОДЕРЖИМОГО =====/
[CONTENT]
SELECT *
FROM some_table
//===== КОНЕЦ СОДЕРЖИМОГО =====/

Флаг [CONTENT] обязательно должен быть в первом абзаце ячейки.

Скрипт может также содержать автозаменяемые поля, например:
    
    SELECT /*$RoadName*//*$*/ -- будет заменено на: SELECT название_дороги
    SELECT '/*$RoadName*//*$*/' -- будет заменено на: SELECT 'название_дороги'
    SELECT '/*$RoadName*/Иркутск/*$*/' -- будет заменено на: SELECT 'название_дороги', содержимое поля (в данном случае Иркутск) бесследно пропадет.

Дополнительные флаги:
[WITH_COUNTER] -- добавить дополнителный первый столбец со счетчиком
[USE_ZERO_COUNTER] -- используется с флагом [WITH_COUNTER], устанавлвивает отсчет счетчик с 0
[AUTO_EXPAND] -- если стоит этот флаг то программа будет добавлять в таблицу дополнительные колонки, если результат скрипта не влезает в таблицу.
[REPLACE_TABLE_ON_EMPTY:СООБЩЕНИЕ_В_СЛУЧАЕ_ОТСУТСТВИЯ_РЕЗУЛЬТАТОВ_ЗАПРОСА] -- заменит таблицу на текст после двоеточия, если запрос не возвращает строк

//===== ПРИМЕР С НЕСКОЛЬКИМИ ФЛАГАМИ =====/
[CONTENT][WITH_COUNTER][REPLACE_TABLE_ON_EMPTY:Записи отсутствуют]
SELECT *
FROM some_table
//===== КОНЕЦ ПРИМЕР С НЕСКОЛЬКИМИ ФЛАГАМИ =====/
